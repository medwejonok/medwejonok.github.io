React приложение для Антифрода

Это React-приложение разработано для коммерческого использования. Приложение помогает выявлять мошенников в определенной отрасли.

Все приложение написано на React. В ходе выполнения были задействованы навыки работы с формами, хуками, роутингом, стилингом, взаимодействием с сервером и дополнительными библиотеками.
Инструмент разметки данных для машинного обучения

Это приложение для разметки данных, разработанное для помощи в создании датасетов для машинного обучения. В его функционале есть возможность работы с изображениями, а также загрузки и скачивания файлов.

Для работы с изображениями был использован canvas, а для обеспечения асинхронности - promise, async, await.
